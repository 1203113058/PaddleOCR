## 五、试样行定位规则

试样条数不固定，需动态扫描定位，规则如下：

1. **扫描范围**：从 `试样编号` 标题行（行 29）的下一行，到 `备注` 行（行 35）的上一行（即行 30 ～ 行 34）
2. **定位方式**：在列 `A` 逐行扫描，遇到非空值即为一条试样的起始行
3. **每条试样占 2 行**（如 22-9637 占行 31-32，22-9638 占行 33-34）
4. **实测值读取列**（取每条试样的第一行）：

| 字段 | 读取列 |
|------|--------|
| 试样编号 | A |
| 屈服强度 | F |
| 抗拉强度 | G |
| 断后伸长率 | I |
| 断面收缩率 | L |
| 冲击试验温度 | P |
| 冲击吸收能量 | R（取试样第1行-R 与第2行-R 拼接，如 `120/124` + `/124` = `120/124/124`） |
| 硬度 (HBW) | W |

> 冲击吸收能量跨两行（如 `120/124` 在行31-R，`/124` 在行32-R），读取时将两行值直接拼接，得到完整的三次测量值 `120/124/124`。
